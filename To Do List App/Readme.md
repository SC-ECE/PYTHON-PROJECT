# GUI Based To Do List App using Tkinter Python

## Description

This is a GUI Based To do List App where users can list their future tasks what needs to be done and there are several other operations like

- Add Task
- Delete One
- Delete All
- Choose Random
- Sort A-Z
- Sort Z-A
- Number of tasks
